 ### Architecture

 ```
   Browser (:8080/ui) ─► proxy (:8080) ─► host (:3000)
                                             ├── /ui              → oe-drilldown.html
                                             ├── /ui/data/*       → dist/*.json
                                             ├── /ui/api/bls      → BLS API proxy
                                             ├── /ui/canvas       → legacy SVG canvas
                                             └── /ui/svg          → push_svg endpoint                                                                                                                                            
 ```
