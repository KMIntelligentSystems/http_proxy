### 1. Prepare a spec and data in Python                                                                                                                                                                                         
                                                                                                                                                                                                                                  
 Inside execute_python, write the .spc file (and any data files it references) to a working folder. X‑13 expects at least 3 full years of monthly data (36 observations) and the command must be given without the .spc
 extension.

 ```python
   import os, textwrap, subprocess, json, tempfile                                                                                                                                                                                
   import requests

   # 1. Fetch or assemble a monthly series (BLS API, CSV, etc.)                                                                                                                                                                   
   series = [...]  # ensure >=36 points, latest-> earliest or vice versa per your spec                                                                                                                                            
   start_year, start_period = 2015, 1  # adjust to match the series

   tempdir = tempfile.mkdtemp()
   basename = os.path.join(tempdir, "myseries")  # will create myseries.spc

   spec = textwrap.dedent(f"""                                                                                                                                                                                                    
   series{{
     start={start_year}.{start_period}
     period=12
     data=({ " ".join(str(v) for v in series) })
   }}
   transform{{ function=log }}
   x11{{ }}
   """)

   with open(basename + ".spc", "w") as f:
       f.write(spec)                                                                                                                                                                                                              
                                                                                                                                                                                                                                  
   # 2. Run X-13ARIMA-SEATS (note: pass basename, no .spc)
   proc = subprocess.run(
       [r"C:/Program Files/x13as/x13as.exe", basename],
       cwd=tempdir,
       capture_output=True,
       text=True,
   )

   if proc.returncode != 0:                                                                                                                                                                                                       
       raise RuntimeError(proc.stdout or proc.stderr)

   # 3. Read desired outputs (e.g., D11 = seasonally adjusted series)
   with open(basename + ".d11") as f:
       sa_lines = [line.strip() for line in f if line.strip()]

   # Parse the .d11 format. Example: first line is header, remaining lines are yyyy.mm value
   from datetime import date
   sa = []
   for row in sa_lines[2:]:
       year, month, value = row.split()                                                                                                                                                                                           
       y, m = map(int, year.split('.'))
       sa.append({"date": f"{y}-{int(float(month)):02d}", "value": float(value)})                                                                                                                                                 

   print(json.dumps({"seasonally_adjusted": sa}))
 ```

 The JSON you print becomes the tool’s response; those values can then feed directly into a D3 visualization.

 ### 2. Key details to remember

 - Path: Always call C:/Program Files/x13as/x13as.exe (forward slashes are fine).                                                                                                                                                 
 If you’d rather shorten it, append the directory to PATH before running your script.
 - Basename, not .spc: X‑13 insists you pass the file stem (e.g., myseries, not myseries.spc). Run from the directory containing the spec or set cwd=tempdir as in the example.
 - Outputs: X‑13 will create files alongside the spec—.d11 for seasonally adjusted series, .d10 for trend, .err for diagnostics, etc. Read whichever you need and return them as JSON.
 - Diagnostics: Always check the .err file; if it contains anything, surface it in your response so you know whether the adjustment succeeded.

 ### 3. Feed into D3

 Once execute_python returns the JSON, you can store it (e.g., write to data/x13_sa.json) or pass it straight into your rendering script. Use the normal Pi workflow:

 1. playwright_navigate → http://localhost:8080/ui.
 2. Prepare an SVG (axis labels, title, source attribution).
 3. Call push_svg with the chart markup generated from your X‑13 results.
 4. Optionally take a playwright_screenshot to validate.
