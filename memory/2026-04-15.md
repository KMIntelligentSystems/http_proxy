## OEWS → BLS Time Series Mapping
- **Workbook in use:** `data/national_M2024_dl.xlsx` (sheet `national_M2024_dl`, 1,403 rows, national scope — AREA=99, AREA_TYPE=1).
- **Key columns available per row:** `AREA`, `AREA_TYPE`, `NAICS`, `I_GROUP`, `OWN_CODE`, `OCC_CODE`, plus employment/wage statistic columns (e.g., `TOT_EMP`, `H_MEAN`, `A_MEDIAN`).
- **Metadata files required for full automation:**
  - `oe.areatype` — numeric `AREA_TYPE` → areatype letter (e.g., 1→`N`, 2→`S`, 4→`M`, 6→`B`).
  - `oe.area` — 7-digit area codes (e.g., national → `0000000`).
  - `oe.industry` — maps (`NAICS`, `I_GROUP`, `OWN_CODE`) to 6-digit `industry_code` (cross-industry → `000000`).
  - `oe.datatype` — two-digit `datatype_code` for each statistic.
  > Note: direct downloads from `https://download.bls.gov/pub/time.series/oe/` returned HTTP 403; manual retrieval or cached copies needed.

## Series ID Construction
```
positions 1–2   survey          "OE"
position 3      seasonal        "U" (OEWS is unadjusted)
position 4      areatype letter (from `AREA_TYPE` via `oe.areatype`)
positions 5–11  area_code       7-digit code (from `AREA` via `oe.area`)
positions 12–17 industry_code   6-digit code (from `oe.industry`)
positions 18–23 occupation      `OCC_CODE` without dash (6 digits)
positions 24–25 datatype        2-digit code (from `oe.datatype`)
```
