## Notes
- Brave `web_search` MCP server returns "Server not initialized" if the Pi CLI/TUI client has not been restarted after configuring credentials.
- Restarting the Pi CLI/TUI (`npm run start:tui` or `npm run dev:tui`) re-establishes the MCP handshake and resolves the initialization error.
